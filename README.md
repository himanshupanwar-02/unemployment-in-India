# unemployment-in-India
This project analyzes monthly unemployment data across Indian states, focusing on unemployment rate, employment levels, and labour participation. It highlights state-wise trends, urban–rural differences, and overall labour market patterns to understand economic conditions.
Unemployment Analysis in India
This project analyzes unemployment trends across Indian states using monthly data. It explores key indicators such as unemployment rate, employment estimates, and labour participation rates, with a focus on regional and rural–urban patterns.
# Features


Data cleaning and preprocessing


Exploratory Data Analysis (EDA)


Trend analysis of unemployment across states and months


Insights based on labour participation and employment


Visualizations for better understanding

# Dataset
The dataset includes:


Region


Date


Frequency


Estimated Unemployment Rate (%)


Estimated Employed


Estimated Labour Participation Rate (%)


Area (Rural/Urban)


# Tools & Libraries


Python (Pandas, NumPy, Matplotlib/Seaborn)


Jupyter Notebook


# Objective
To understand unemployment patterns in India and derive insights that can help in socio-economic assessment and policymaking.
